> このプラグインは現在**開発中**です。

[![.NET](https://img.shields.io/badge/.NET-10.0-purple.svg)](#)
[![Release](https://img.shields.io/github/v/release/routersys/YMM4-CustomStrokeForge.svg)](https://github.com/routersys/YMM4-CustomStrokeForge/releases)

YukkuriMovieMaker4 (YMM4) 上で動作する、**高品質カスタム縁取り映像エフェクトプラグイン**です。
Jump Flood Algorithm（JFA）による符号付き距離場（SDF）計算をHLSLカスタムピクセルシェーダーで実装し、実線・点線・破線・鎖線・二重線・波線の6種類の縁取りスタイルを、滑らかなエッジ処理（ソフトネス）付きで描画します。
縁取りの位置は外側・中央・内側から選択でき、全パラメーターがYMM4のアニメーションシステムと統合されているため、キーフレームによる動的な縁取り演出が可能です。

## 概要

本プラグインは、YMM4の映像エフェクト（コンポジションカテゴリ）として動作するカスタム縁取りエフェクトです。
Direct2Dカスタムシェーダー（`D2D1CustomShaderEffectBase`）を基盤とし、HLSLピクセルシェーダーによるGPUレンダリングで高品質な縁取りを実現します。
内部処理はJFAによる距離場計算を3フェーズ（シード初期化・JFA伝播・描画）に分けて実行しており、対数ステップ数の伝播パスに加えて2パスの精度補正を行うことで、高精度な距離推定を達成しています。
なお、本エフェクトはAviUtlエクスポートには対応していません。

## 主な機能

### 1. 6種類の縁取りスタイル

| スタイル | 説明 |
|---|---|
| **実線** | 連続した実線の縁取り |
| **点線** | 点で構成された縁取り（破線長と間隔で制御） |
| **破線** | 短い線分を繰り返す縁取り（破線長と間隔で制御） |
| **鎖線** | 長短の線分が交互する縁取り（破線長・間隔・短破線長で制御） |
| **二重線** | 内側に間隔を設けた二重平行線の縁取り（二重線間隔で制御） |
| **波線** | シェーダー上でsin関数を用いた波形変位による縁取り（振幅・周波数で制御） |

### 2. 3種類の縁取り位置

- **外側**: 図形の外側へ縁取りを描画します。アルファ値が0の領域はマスクされます。
- **中央**: エッジに沿って中央に縁取りを描画します。縁取り幅は実効値の0.5倍として処理されます。
- **内側**: 図形の内側へ縁取りを描画します。距離場の符号を反転させて処理します。

### 3. 調整可能なパラメーター

#### 縁取りグループ
- **線幅**: 縁取りの太さを0.5〜50pxの範囲で設定します。アニメーション対応。
- **色**: 縁取りの色をカラーピッカーで指定します。
- **不透明度**: 縁取りの不透明度を0〜100%の範囲で設定します。アニメーション対応。
- **位置**: 外側・中央・内側から選択します。

#### パターングループ
- **破線長**: 破線・点線・鎖線の破線部分の長さ（1〜200px）。アニメーション対応。
- **間隔**: 各セグメント間の隙間の長さ（1〜200px）。アニメーション対応。
- **短破線長**: 鎖線の短い破線部分の長さ（1〜100px）。アニメーション対応。
- **二重線間隔**: 二重線スタイル時の線間の間隔（0.5〜30px）。アニメーション対応。
- **波の振幅**: 波線の振れ幅（0〜50px）。アニメーション対応。
- **波の周波数**: 波線の周波数（0.1〜100）。アニメーション対応。
- **位相**: パターンアニメーション用の位相オフセット（-1000〜1000）。アニメーション対応。

#### 詳細グループ
- **ソフトネス**: 縁取りエッジの柔らかさ（0〜10）。アニメーション対応。シェーダー内で`smoothstep`により適用されます。

### 4. アニメーション対応

線幅・不透明度・破線長・間隔・短破線長・二重線間隔・波の振幅・波の周波数・位相・ソフトネスの全10パラメーターがYMM4のアニメーションシステムと統合されており、タイムライン上でキーフレームアニメーションを設定できます。

### 5. タブ式エディタUI

エフェクトのプロパティパネルは「縁取り」「パターン」「詳細」の3タブに整理されており、用途に応じたパラメーターに素早くアクセスできます。

### 6. シェーダーベースのGPUレンダリング

エフェクト処理はすべてDirect2Dカスタムシェーダー（`ps_5_0`）による GPU 上で実行されます。
描画バッファ（`R32G32B32A32_Float`形式）は入力解像度に応じて動的にサイズが決定され、必要な場合のみ再生成されます。

## 技術的な動作概要

本エフェクトは以下の3フェーズでレンダリングを行います。

1. **シード初期化**: 入力テクスチャの各ピクセルのアルファ値をしきい値（0.01）と比較し、図形の輪郭ピクセル座標をシードとして書き出します。
2. **JFA距離伝播**: `log2(max(width, height))` ステップの対数スケール伝播パスと、精度補正のための2回の固定パス（`stepWidth = 1.0`）を実行し、各ピクセルから最近傍シードまでの距離場を構築します。
3. **縁取り描画**: 距離場からスタイル・位置・ソフトネスを考慮したマスクを生成し、縁取りを描画します。

## 動作要件

- **OS**: Windows 10 / 11 (64bit)
- **YukkuriMovieMaker4**: 最新版を推奨
- **ランタイム**: .NET 10.0
- **その他**: Direct2D / Direct3D 11 対応GPU

## 使い方ガイド

### 基本的な使用フロー

1. YMM4のタイムライン上で映像アイテムを選択し、エフェクトの追加を行います。
2. エフェクトのプロパティパネルにタブ式エディタが表示されます。
3. **「縁取り」タブ**で線幅・色・不透明度・位置を設定します。
4. **「パターン」タブ**で線種に応じた破線長・間隔・波パラメーターを調整します。
5. **「詳細」タブ**でソフトネスを調整し、エッジの滑らかさを制御します。
6. 各パラメーターにYMM4のキーフレームを設定することで、アニメーションした縁取りを表現できます。

### スタイル別パラメーター参照

| スタイル | 有効なパラメーター |
|---|---|
| 実線 | 線幅・色・不透明度・位置・ソフトネス |
| 点線 | 上記 + 破線長・間隔 |
| 破線 | 上記 + 破線長・間隔 |
| 鎖線 | 上記 + 破線長・間隔・短破線長 |
| 二重線 | 線幅・色・不透明度・位置・ソフトネス + 二重線間隔 |
| 波線 | 上記（実線相当） + 波の振幅・波の周波数・位相 |

## 注意事項

- 本エフェクトは**AviUtlエクスポートには対応していません**。
- 入力映像のサイズが大きいほど、JFA伝播のステップ数（`log2(max(width, height))`）が増加し、GPU負荷が上昇します。
- パターン系スタイル（点線・破線・鎖線）の位相パラメーターは、シェーダー内でシード座標の角度と半径を用いた座標系に基づいて計算されるため、直線的なパターン間隔とは異なる挙動を示す場合があります。
- **インターネット通信**: 本プラグインはインターネット通信を行いません。
